**System:** Pathfinder 2e

## Session Notes

Session logs are in `knightly-notes/`, named by date. Each file covers one session and includes combat notes, NPC interactions, decisions made, and items acquired.

---

## Tools & Workflow

Notes are written in Markdown and managed as an [Obsidian](https://obsidian.md) vault. Internal links use `[[wiki-link]]` syntax.
